## SmartHosts
Merge hosts configs files into hosts via merge_tool.sh~
（维护可用的hosts配置文件，通过merge_tool快速生成和配置到自己的电脑上~）

### USAGE

```
git clone git@github.com:IceskYsl/smart_hosts.git
cd smart_hosts
chmod +x merge_tool.sh
sudo ./merge_tool.sh
```
